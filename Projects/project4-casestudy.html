Case Study: Music Mood Prediction with Deep Learning

The Context

Music platforms and applications increasingly rely on automated mood detection to enhance user experience, recommend playlists, and support creative analysis. Accurately classifying the mood of a song requires understanding complex audio patterns and extracting meaningful features.

The Objective

Develop a machine learning system that classifies songs into four moods — sad, happy, energetic, or calm — based on numerical audio features extracted from each track. The system should be robust, scalable, and capable of learning intricate patterns from real-world music data.

The Approach

I implemented a neural network classifier in PyTorch, leveraging audio features such as danceability, energy, speechiness, liveness, valence, tempo, and duration. Preprocessing included duplicate removal, log-transform for skewed features, outlier filtering, winsorization, and standard scaling. The network architecture consisted of two hidden layers (64 and 32 neurons) with ReLU activation and dropout, optimized using Adam with cross-entropy loss. The model was trained using mini-batch gradient descent over 30 epochs. 

class NeuralNet(nn.Module): 
    def __init__(self, input_size=9, hidden1=64, hidden2=32, output_size=4, dropout=0.3): 
        super(NeuralNet, self).__init__() 
        self.fc1 = nn.Linear(input_size, hidden1) 
        self.dropout1 = nn.Dropout(dropout) 
        self.fc2 = nn.Linear(hidden1, hidden2) 
        self.dropout2 = nn.Dropout(dropout) 
        self.fc3 = nn.Linear(hidden2, output_size) 
         
    def forward(self, x): 
        x = F.relu(self.fc1(x)) 
        x = self.dropout1(x) 
        x = F.relu(self.fc2(x)) 
        x = self.dropout2(x) 
        x = self.fc3(x)  # logits 
        return x

Results & Performance

Model Accuracy
80% Test Accuracy

The model achieved approximately 80% accuracy, demonstrating its capability to reliably classify songs. Loss curves and a confusion matrix indicate balanced performance across the four mood classes.

Architecture

The neural network utilized two hidden layers (64 and 32 neurons) optimized using Adam with cross-entropy loss, trained over 30 epochs using mini-batch gradient descent.
